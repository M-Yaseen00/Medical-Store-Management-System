# Medical Store Management System

A console-based C# application for managing patient, doctor, and staff entities in a medical store.

![Application Screenshot](screenshot.png)

## Features

- Manage patient records, including their personal information and medical history.
- Keep track of doctor details, including their specialization and contact information.
- Handle staff records for the medical store.
- Perform operations like adding, updating, and deleting records.
- Generate reports and summaries of patient data.
- Simple and easy-to-use console interface.

## Getting Started

### Prerequisites

- .NET Framework (version X.X.X)
- Visual Studio (or your preferred C# development environment)

   ```bash
   git clone https://github.com/M-Yaseen00/medical-store-management.git

   Open the project in Visual Studio or your preferred IDE.

Usage
Launch the application in your console or terminal.

Follow the on-screen prompts to perform various tasks, such as managing patient records, doctor information, and staff details.

Use the menu options to navigate through different functionalities.

Generate reports and summaries as needed.
